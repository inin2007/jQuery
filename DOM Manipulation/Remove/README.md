# remove()
It removes the selected element (and its child elements).

# empty() 
It removes the child elements from the selected element.

## Example:
remove()
```
$("p").remove();
```
empty()
```
$("p").empty();
```
# Filter the Elements to be Removed
```
$("h2").remove(".img");
```
Or
```
$("h2").empty(".img");
```
